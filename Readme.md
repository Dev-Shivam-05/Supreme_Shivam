## 🏛️ Technical Architecture Breakdown

The portfolio architecture is designed for **maximum performance (98+ Lighthouse score)** and **seamless user experience**. It follows a modular, component-based approach within a vanilla ecosystem to demonstrate deep fundamental mastery.

### **Core Architecture Layers:**
1.  **Presentation Layer (UI/UX):**
    -   **Engine:** HTML5 Semantic Markup & CSS3 (Custom Properties / Flexbox / Grid).
    -   **Motion Design:** GSAP (GreenSock Animation Platform) for high-performance, frame-rate independent animations.
    -   **Smooth Scrolling:** Lenis for optimized inertia-based scrolling and unified scroll experience across browsers.
2.  **Logic Layer:**
    -   **Core:** Modern ES6+ JavaScript.
    -   **State Management:** Localized object-based state for project rendering and UI transitions.
    -   **Integration:** RESTful communication with Web3Forms for serverless lead generation.
3.  **Data Layer:**
    -   **Structured Metadata:** JSON-LD for rich-snippet generation and advanced SEO.
    -   **Dynamic Content:** Data-driven rendering for projects and technical stack to ensure maintainability.
