The Problem

Existing AI art tools were either prohibitively expensive, locked behind complex interfaces, or lacked community features. Users wanted a simple, affordable platform where they could generate AI art from text prompts, save creations to a personal gallery, and share with a community — all without needing a PhD in machine learning.

The Solution

I built a full-stack MERN application with a clean, intuitive UI. The backend leverages Node.js and Express to proxy requests to OpenAI's DALL·E API, while MongoDB stores user data and generated artworks. JWT-based authentication, image optimization via Cloudinary, and a responsive React frontend make it production-ready and blazing fast.
